Input notation
l or ell→ ℓ (loop momentum)
l1, l2→ ℓ1, ℓ2
p, p1, p2→ p, p1, p2
s12, s23→ s12, s23 (Mandelstam)
m_H, M_W, m1→ mass labels (use _ for subscript)
l.p→ ℓ · p (dot product → CenterDot)
l*p→ ℓ p (scalar multiplication)
(l+p)^2→ (ℓ + p)²
1/2→ ½ (fraction)
-> or =→ → (substitution rule)
Output variables
mm, mm1, mmH= internal (propagator) mass²
MM, MM1, MMW= external momentum squared
s12, s23= Mandelstam invariants
Mass label mapping: m_H → output variable mmH; M_W → MMW.
Use output variable names in substitution rules, e.g. {mmH -> 1, s12 -> 6}.
Custom momentum & mass labels from your diagram are recognized automatically.

Keyboard Shortcuts

        

          

            
Drawing

            
Click Place vertex

            
Drag Create edge between vertices

            
Shift+Drag Move vertex

            
Click edge Subdivide edge

            
Drag → return Create tadpole (self-loop)

          

          

            
Modes

            
1 Draw mode

            
2 Delete mode

            
N Cycle labels (None / Numbers / Momenta)

            
S Toggle snap to grid

          

          

            
Navigation

            
Ctrl+Drag Pan canvas

            
Ctrl+Scroll Zoom in/out

            
+ / − Zoom in/out

            
0 Reset zoom

          

          

            
Actions

            
Ctrl+Z Undo

            
Ctrl+Y Redo

            
L Auto-arrange layout

            
C Toggle configuration panel

            
Tab Cycle selected vertex

            
Esc Deselect / close panel

          

          

            
Edge Properties

            
Click the small circle on any edge to set mass, line style, particle label, or momentum direction.
